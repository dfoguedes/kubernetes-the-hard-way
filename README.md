# kubernetes-the-hard-way

This repository follows the "Kubernetes the Hard Way" tutorial, automating the setup of a Kubernetes cluster from scratch using Vagrant for educational purposes.

## Repository Structure

- `Vagrantfile`: Vagrant configuration for provisioning local VMs.
- `scripts/`: Contains setup and automation scripts.

## Getting Started

1. Install [Vagrant](https://www.vagrantup.com/) and a supported provider (e.g., VirtualBox).
2. Clone this repository and run:
   ```sh
   vagrant up
   ```
